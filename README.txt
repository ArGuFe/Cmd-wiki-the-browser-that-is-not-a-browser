=====Objective=====

The objective of the program is to reduce the memory used by other browsers.
You can use this "browser" to search things and download by url.

=====How to open it=====

To open it you need to:

1. Open the folder which the "main.py" file is.

2. Click in the folder search bar.

3. Type in "cmd" in the folder search bar.

4. In the command prompt type "main.py".

5. Done!
